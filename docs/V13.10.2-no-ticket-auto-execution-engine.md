# AlphaPilot V13.10.2 - No-Ticket Auto Execution Engine

V13.10.2 merges two requested changes:

1. Auto-execution lifecycle and strategy arbitration.
2. No user-facing ticket workflow.

The goal is to make the local pre-live flow closer to the desired future
automation path:

```text
public market candidates
-> strategy router
-> local risk gate
-> local lifecycle record
-> review and replay
```

It does not submit Demo or live exchange orders.

## What Changed

- Added `GET /api/auto-execution-engine`.
- Added `POST /api/auto-execution-engine/run`.
- Added local state lists:
  - `autoExecutionRuns`
  - `autoExecutionRecords`
- Added a strategy router that:
  - groups candidates by symbol
  - resolves same-symbol direction conflicts
  - applies an open-record cooldown
  - caps a run to five selected records
- Added a local risk gate for:
  - public market readiness
  - target R >= 2
  - score >= 50
  - trade count >= 5
  - profit factor >= 1
  - notional <= 1000 USDT
- Replaced the main no-key page action from “create observation ticket” to
  “run auto execution engine”.

## Boundary

The new records are local simulation lifecycle records.

They are not:

- exchange orders
- OKX Demo orders
- live orders
- account reads
- position reads
- private API calls
- withdrawal calls

Each record explicitly stores:

- `apiKeyUsed: false`
- `ordersCreated: false`
- `demoOrderCreated: false`
- `liveTrading: false`
- `withdrawEnabled: false`
- `userFacingTicketWorkflow: false`

## Why Keep Internal Records

The user-facing ticket step is removed, but immutable internal records remain
necessary for review and safety:

- which strategy was selected
- which symbol was selected
- which router rule won
- which risk gate passed or blocked
- which TP/SL policy was attached
- when the lifecycle record was created

This keeps the flow fast without losing auditability.

## Still Locked

V13.10.2 does not unlock:

- raw API key storage
- Trade API
- Withdraw API
- real account reads
- real position reads
- Demo order submission from the auto engine
- live order submission
- unattended live automatic trading
