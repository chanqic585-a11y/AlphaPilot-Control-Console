# V13.9.8 Navigation Simplification and Mobile Console Page

V13.9.8 simplifies the desktop control console navigation after the V13.9.7
page split.

## Main Navigation

The left navigation is now limited to five operator-facing pages:

1. `策略`
2. `本地模拟`
3. `Demo模拟`
4. `实盘交易`
5. `手机控制台`

This avoids mixing primary workflow pages with older research, audit, public
probe, strategy slot, and debug panels.

## Mobile Console Page

`手机控制台` is now a standalone page with the same hash-routing behavior as the
other four primary pages.

It shows:

- recommended mobile URL
- local status API URL
- connection notes
- read-only mobile status payload
- safety boundary for mobile usage

The page is read-only. It does not save API keys, create orders, or enable live
trading.

## Advanced Modules

Advanced research modules remain in the codebase and can still be displayed via
the advanced research toggle. They are no longer listed as primary navigation
items.

This preserves diagnostics without making the daily control surface noisy.

## Safety Boundary

This patch does not enable:

- Trade API
- Withdraw API
- raw API key storage
- real account access
- real position access
- real order creation
- automatic trading
- automatic Demo order submission
