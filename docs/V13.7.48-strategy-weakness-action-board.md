# AlphaPilot V13.7.48 - Strategy Weakness Action Board

V13.7.48 adds a local research action layer on top of V13.7.47 replay scoring.

## Purpose

The previous version labels replay weaknesses such as deep adverse excursion,
profit not captured, stop-like loss, cost drag, and holding too long.

This version turns those weakness labels into research action items:

- what weakness appeared
- which strategy it affects
- how many samples show it
- whether it blocks upgrade
- what research task should be done next

## Added API

```text
GET /api/weakness-action-board
GET /api/weakness-action-board/actions
```

The endpoints read local closed-sample replay summaries and return deterministic
research actions. They do not modify strategy logic or generate new sandbox
samples.

## Web Console

The console now shows a `策略弱点行动板` panel after the closed-sample replay
panel. It displays:

- action count
- high-priority action count
- medium-priority action count
- blocked-upgrade count
- top research actions
- research tasks for each weakness

## Safety Boundary

This version is local research only.

It does not save API keys, connect Trade API, connect Withdraw API, read real
accounts, read real positions, create orders, run exchange Dry-run, or enable
automatic trading.

The action board is not a trading signal and is not trading advice.
