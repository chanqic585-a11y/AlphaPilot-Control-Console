# V13.27.1.1 Dual-Data One-Click Backtest

## Purpose

V13.27.1.1 connects the Strategy page to the Quant Engine dual-layer data and
backtest workflow. It preserves one immutable strategy version while allowing
data-preparation retries to create new evaluation attempts.

The workflow order is:

```text
Read-only local research smoke
-> official OKX public data preparation
-> immutable snapshot and validation manifests
-> fixed targetR >= 2 formal backtest
-> awaiting Local Forward only after a formal pass
```

## User Interface

The Strategy card exposes one `一键回测` action and a compact progress view:

- current workflow phase;
- research or formal evidence class;
- required and completed public-data partitions;
- funding-data coverage;
- snapshot and validation status;
- failure or blocker summary;
- automatic next stage, which is limited to Local Forward.

Internal hashes, raw paths, and detailed evidence remain under advanced
details. The UI requests work through Quant Engine; it cannot write a passed
status itself.
