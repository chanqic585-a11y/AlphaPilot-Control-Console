# AlphaPilot V13.7.36 - Forward Review Workbench

V13.7.36 adds a desktop-console workbench for the July 10 forward-review window.

The workbench is review-only. It does not connect to exchanges, store API keys, read real accounts, read real positions, create orders, run exchange dry-run, or automate trading.

## What It Shows

- The five current strategy candidates under live-readiness review.
- Real manual forward logs, counted separately from local virtual sandbox replay logs.
- Real closed forward samples, rule matches, risk warnings, invalidations, and latest observation logs.
- Review blockers for each strategy.
- A candidate expansion pool sourced from the existing strategy candidate queue.

## Candidate Expansion

More strategies can become candidates, but V13.7.36 does not auto-promote them.

Expansion candidates must still go through:

1. research queue review,
2. backtest or factor evidence,
3. paper or shadow observation,
4. real forward logs,
5. manual pre-live review,
6. explicit user decision outside this console.

## API

- `GET /api/forward-review`
- `POST /api/forward-review/refresh`

The refresh endpoint re-imports local Quant Engine reports and observation state only. It does not call any private exchange API.

## Safety Boundary

Disabled in this version:

- API key input
- Trade API
- Withdraw API
- real account access
- real position access
- real order creation
- exchange dry-run
- live trading
- automatic trading

V13.7.36 is a review workbench, not a trading system.
