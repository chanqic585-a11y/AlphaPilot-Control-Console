# V13.14.0 Automatic Demo Execution

The Control Console now consumes immutable, checksum-verified Quant Engine
Demo contracts. The automatic path is:

```text
eligible DemoRelease
  -> signal arbitration
  -> 1000 USDT Demo risk envelope
  -> append local intent
  -> OKX Demo order with clOrdId
  -> query partial or final fills
  -> audit, drift pause, or kill switch
```

No individual ticket is required after a release and its fixed risk envelope
are enabled. Runtime startup remains an explicit local action. Missing release,
missing process credentials, any disabled gate, checksum mismatch, stale data,
liquidity failure, risk breach, unknown order state, or ledger mismatch stops
new entries.

The old manual Demo rehearsal remains available only in advanced mode for
compatibility. It is not part of the automatic release path.

Live remains locked. V13.14.0 does not create a Live release and cannot call a
Withdraw endpoint.
