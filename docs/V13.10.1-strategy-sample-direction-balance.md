# AlphaPilot V13.10.1 - Strategy Sample and Direction Balance Polish

V13.10.1 improves the no-key pre-live workbench without adding any trading
capability.

## Why This Patch Exists

After V13.10.0, the console could show new no-key pre-live candidates while old
local sandbox samples were still stored in the state file. That made it easy to
mistake a new candidate layer for missing historical samples.

The public candidate list was also dominated by short-cycle short strategies
because candidates were selected by strategy score order first. The usable
strategy catalog already contains both short and long-research strategies, so
the workbench now shows that direction balance explicitly.

## What Changed

- Added `sampleLayerSummary` to `/api/no-key-pre-live`.
- Added `directionBalance` to `/api/no-key-pre-live`.
- Added `universeScope` to explain that current scans use historical
  `selectedPairs`, not full-market scanning.
- Added `longCandidateLane` to explain the long-side research lane.
- Changed no-key candidate generation to `balanced_strategy_first`, giving each
  strategy a first candidate before adding extra symbols.
- Increased the no-key public scan request to 12 candidates so the first pass
  can include both short and long-research strategies.
- Updated the web console to show sample layer, direction balance, universe
  scope, and long-candidate lane cards.

## Current Scope

The current scanner is still a public-market candidate scanner. It does not
scan the full OKX market yet. It uses the strategy catalog's selected pairs and
checks public market availability.

The next natural step is a liquidity-filtered market-wide scanner:

1. Load OKX public instruments and tickers.
2. Filter by liquidity, quote asset, data availability, and market type.
3. Run short and long candidate rules over the eligible public universe.
4. Keep all outputs as local research candidates until Demo gates are ready.

## Safety Boundary

V13.10.1 does not:

- require API keys
- store raw API keys
- call private exchange endpoints
- read real accounts
- read real positions
- create Demo orders
- create live orders
- enable automatic trading

Observation tickets remain local review records only. They are not exchange
orders and are not trading instructions.
