# AlphaPilot Control Console V13.26.0

V13.26 adds a local, immutable bridge from complete Demo/Live trade outcomes to
the Quant Engine offline feedback loop.

## Recording rule

The internal execution engine can register an outcome only when the opening
execution record exists and is filled, and the caller supplies explicit close
evidence. The ledger verifies ordered timestamps, finite values, positive risk,
and `netPnl = grossPnl - feePaid - slippagePaid`.

Live records also bind the exact LiveRelease and RiskProfile ids/checksums.
There is no browser endpoint for submitting arbitrary outcome payloads.

## Export rule

`POST /api/execution-outcomes/export` writes a checksummed local JSON export.
Opening fills, non-terminal records, canceled/rejected entries, and any record
without complete close evidence remain listed as quarantined.

The export does not contain credentials or account values, does not update a
model, does not promote a release, and does not create an order.
