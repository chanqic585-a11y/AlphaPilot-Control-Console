# V13.8.9 Testnet Local Drill

V13.8.9 adds a local Testnet Drill panel to the AlphaPilot Control Console.

The goal is to make the next pre-testnet step visible without enabling exchange
execution. The console now shows which strategies are only observation items,
which ones are local review candidates, and which ones would eventually be
eligible for a future testnet drill review.

## Added

- `GET /api/testnet-drill`
- Web console panel: `Testnet 本地演练`
- 1,000 USDT virtual account drill template
- Local order lifecycle drill steps
- Local risk template:
  - single strategy virtual account: 1,000 USDT
  - single risk: about 1R / 1%
  - reward-risk target: 2R, not reduced
  - daily stop: -2R
  - strategy pause on abnormal state or data gaps
- Local rehearsal summary using existing pre-live rehearsal records
- A button that saves one local lifecycle rehearsal only

## Data Flow

```text
Strategy Asset Playbook
-> Testnet Drill Builder
-> Existing Pre-live Rehearsal Records
-> Web Console Testnet Local Drill Panel
```

The drill button writes only to the local pre-live rehearsal store. It does not
call any exchange endpoint.

## Safety Boundary

V13.8.9 does not add:

- API key input
- API key storage
- Trade API
- Withdraw API
- private exchange connection
- real account reads
- real position reads
- order creation
- exchange Dry-run
- live trading
- automatic trading

The drill is a local audit artifact only. It is not a trading command.
