# AlphaPilot Control Console V13.24.0

The Live page now includes a compact Chinese RiskProfile panel for reviewing,
versioning, activating, and rolling back bounded portfolio settings.

- Saving creates a new immutable version.
- Activation requires an exact confirmation phrase and creates an audit event.
- Live environments require `ACTIVATE_LIVE_RISK_PROFILE`.
- Rollback selects the previous distinct immutable version.
- The shared portfolio evaluator blocks strategy, position, symbol, direction,
  correlation, loss, drawdown, cooldown, stale-data, and liquidity breaches.
- A profile id/hash mismatch blocks release execution.

These controls do not enable Demo or Live trading. V13.24 has no Live exchange
adapter, stores no raw credentials, and includes no Withdraw capability.
