# AlphaPilot V13.6.6 - Desktop and Mobile Command Console UI Polish

V13.6.6 improves the visual interface of the desktop Control Console and aligns the mobile App control-console page with the same Chinese-first dark quant terminal style.

## Desktop UI

- Uses a labeled left navigation rail instead of icon-only navigation.
- Keeps the main cockpit focused on strategy, signal, simulated position context, and backtest result summaries.
- Improves the one-screen matrix so the current research state is easier to scan.
- Keeps public exchange connectivity, strategy slots, mobile bridge, and audit log visible.

## Mobile UI

- Turns the App-side Control Console page into a compact mobile command console.
- Highlights current strategy, signal count, simulated PnL context, backtest metrics, public connectivity, and execution locks.
- Keeps connection diagnostics available without letting them dominate the page.
- Uses explicit simulated / paper wording for position and PnL context.

## Safety Boundary

V13.6.6 does not add:

- API key input or storage.
- Trade API.
- Withdraw API.
- Private exchange endpoints.
- Real account reads.
- Real position reads.
- Order creation.
- Exchange Dry-run execution.
- Live trading.
- Automatic trading.
