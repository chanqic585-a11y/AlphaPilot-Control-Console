# AlphaPilot V13.6.5 - Chinese Quant Terminal UI

V13.6.5 updates the desktop Control Console into a Chinese-first dark quant terminal.

## UI Direction

The page uses a dense research-console layout:

- left rail navigation
- top status bar
- portfolio context summary
- one-screen strategy matrix
- public exchange connectivity panel
- strategy slots
- mobile bridge
- audit log

The main matrix places these four research objects together:

- strategy
- signal
- simulated position context
- backtest result

## Safety Boundary

This is still a research and control console only.

V13.6.5 does not add:

- API key input
- Trade API
- Withdraw API
- real account reads
- real position reads
- order creation
- exchange Dry-run
- live trading
- automatic trading

The simulated-position section only displays local paper context such as max
concurrent simulated positions. It does not read exchange positions.
