# AlphaPilot V13.7.44 - Simulation Review Queue and Strategy Promotion Panel

V13.7.44 turns local sandbox samples into a review queue. It does not add any
exchange permission or trading execution path.

## Goal

Organize existing local simulation samples so each strategy has a clear review
state:

- collecting samples
- under review
- promoted candidate
- watchlist
- demoted
- archived reference

These states are suggestions for human review. They are not written back as
automatic lifecycle transitions.

## Data Sources

The review queue reads only local files:

- usable strategy catalog
- local sandbox daily reports
- paper observation logs
- local sandbox auto-runner state

No backend exchange request, API key, private endpoint, account read, position
read, order creation, exchange Dry-run, live trading, or automatic trading is
used.

## API

- `GET /api/simulation-review`
- `GET /api/simulation-review/strategies`
- `GET /api/simulation-review/queue`
- `GET /api/simulation-review/strategies/{strategyId}`

## Metrics

The queue computes:

- closed sample count
- virtual PnL / equity
- win rate
- profit factor
- average win R
- average loss R
- max consecutive losses
- max drawdown R
- pair breakdown
- direction breakdown when available
- market-regime breakdown when available
- sample-size, concentration, inactive, risk, and invalidated-sample warnings

If direction, regime, fee, or slippage fields are missing, V13.7.44 keeps them
as review gaps. It does not fabricate missing data.

## Gates

Current gates:

- 30 closed samples: human review threshold
- 100 closed samples: later Dry-run sample threshold
- 300 closed samples: stability threshold
- Profit Factor above 1.05: promoted-candidate suggestion threshold
- Profit Factor above 1.15: later Dry-run quality threshold

`dryRunApproved` and `liveTradingApproved` remain `false`.

## Safety Boundary

V13.7.44 is research-only:

- no API key storage
- no Trade API
- no Withdraw API
- no account balance reads
- no position reads
- no order creation
- no exchange Dry-run
- no live trading
- no automatic trading
