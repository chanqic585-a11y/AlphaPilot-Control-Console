# AlphaPilot V13.27.5 - Cancelled Attempt Resume

The strategy page now distinguishes pause from cancellation.

- A queued card shows 暂停排队.
- A paused card shows 继续回测.
- A cancelled card shows 从检查点重新开始.
- Restart creates a new audited attempt and preserves the old attempt.
- Permanent cancellation requires explicit browser confirmation.
- Restarted attempts use the serial batch worker.

This patch does not change strategy parameters, evidence gates, targetR >= 2,
Demo credentials, Live permissions, or order behavior. Credentials remain
process-only and Withdraw remains absent.
